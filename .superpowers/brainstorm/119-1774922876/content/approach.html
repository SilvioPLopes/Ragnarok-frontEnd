Qual nível de "cara de RO" você quer?

Isso define o escopo do rework visual. Escolha a abordagem.

A

Consistência — arruma o que está quebrado

Corrige as inconsistências sem mudar a estrutura. HP bars usando tokens CSS corretos (vermelho, não emerald-500). Tipografia pixel em todos os rótulos. Botões sempre usando pixel-button. Cores hardcoded removidas. O BattleHud já está certo — equaliza o resto com ele.

Escopo: ~1 dia. Zero risco de quebrar layout.

B

RO Theme — identidade visual completa

Além da consistência: borda dupla com bevel nos painéis (estilo janela RO clássica), tab bar com visual de botões quadrados tipo RO (não tabs modernas), stat bars com gradiente 3D, painel de stats com layout de ficha de personagem, botões com inset shadow de relevo.

Escopo: ~2-3 dias. Algumas classes Tailwind customizadas.

C

Full RO Window System — máxima fidelidade

O RO clássico tem janelas com título bar (faixa azul/marrom com texto branco), botão X de fechar, bordas de 3px com highlight/shadow. Cada painel vira uma "janela" do RO. Login recria a tela de login original do iRO/kRO. Tab bar vira os ícones quadradinhos do RO (não texto).

Escopo: ~5 dias. Maior impacto visual, maior risco.

Pode selecionar mais de um se quiser mix. A recomendação é B — o equilíbrio certo entre impacto e praticidade.
